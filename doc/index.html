Mule CoAP connector - CoapServer Userguide

The CoapServer Connector can be used in Mule applications to implement CoAP servers as defined in rfc7252 . A CoAP server is defined by means of a set resources on which requests can be done like GET, POST, PUT etc. . The server CoAP endpoint has a number of configuration parameters that can be used to tune behavior of the server. These parameters have sensible defaults and need only to be set for specific needs.
